# 🖼️ Image Gallery

A simple and elegant image gallery built using **HTML**, **CSS**, and **JavaScript**. This project displays a grid of images that users can click to view in a larger popup format.

## 🚀 Features

- Responsive image grid using CSS Grid
- Popup image preview on click
- Smooth layout with gradient background
- Font Awesome icon integration for close button

## 🛠️ Technologies Used

- HTML5
- CSS3 (with gradient backgrounds and grid layout)
- JavaScript (DOM manipulation, event handling)
- Font Awesome Icons

## 📸 How It Works

- Click any image in the gallery to open a popup with a larger preview.
- Use the close (`x`) icon to return to the gallery view.
